# 双因子组合策略

结合**独立强度因子**和**动量因子**的综合选股策略。

---

## 策略逻辑

```
┌─────────────────────────────────────────────────────────────┐
│                      双因子组合策略                          │
├─────────────────────────────────────────────────────────────┤
│                                                             │
│   ┌──────────────────┐        ┌──────────────────┐         │
│   │  独立强度因子     │        │   动量因子       │         │
│   │  (抗跌能力)      │        │  (上涨动能)      │         │
│   └────────┬─────────┘        └────────┬─────────┘         │
│            │                           │                    │
│            ▼                           ▼                    │
│   ┌──────────────────────────────────────────┐             │
│   │          综合得分 = 权重1 × 独立强度      │             │
│   │                  + 权重2 × 动量           │             │
│   └─────────────────────┬────────────────────┘             │
│                         │                                   │
│                         ▼                                   │
│   ┌──────────────────────────────────────────┐             │
│   │         排序选取得分最高的Top N           │             │
│   └──────────────────────────────────────────┘             │
│                                                             │
└─────────────────────────────────────────────────────────────┘
```

---

## 因子说明

### 因子1: 独立强度因子

- **来源**: `01.independence-score`
- **作用**: 衡量股票在板块下跌时的抗跌能力
- **特点**: 防御性强，选出质地好的股票

### 因子2: 动量因子

- **来源**: `02.momentum-factor`
- **作用**: 衡量股票的上涨动能
- **特点**: 进攻性强，选出趋势好的股票

---

## 权重配置

| 市场环境 | 独立强度权重 | 动量权重 | 说明 |
|---------|-------------|---------|------|
| 牛市 | 30% | 70% | 强者恒强，重动量 |
| 震荡市 | 50% | 50% | 平衡配置 |
| 熊市 | 70% | 30% | 防守为主，重质地 |

---

## 使用方法

### 计算某日双因子得分

```bash
./scripts/calc_combined_factor.py 2026-03-26
```

### 调整权重

```bash
# 牛市配置（动量权重70%）
./scripts/calc_combined_factor.py 2026-03-26 --momentum-weight 0.7

# 熊市配置（独立强度权重70%）
./scripts/calc_combined_factor.py 2026-03-26 --independence-weight 0.7
```

### 回测

```bash
./scripts/backtest_combined_factor.py \
    --start 2026-01-01 \
    --end 2026-03-26 \
    --top-n 20
```

---

## 核心公式

```
综合得分 = (独立强度分数 / 独立强度最大值) × w1 × 100
        + (动量分数 / 动量最大值) × w2 × 100

其中:
- w1 = 独立强度权重 (默认0.5)
- w2 = 动量权重 (默认0.5)
- w1 + w2 = 1.0
```
