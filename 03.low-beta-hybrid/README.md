# 低贝塔混合策略 (Low Beta Hybrid)

**策略编号**: 03  
**策略名称**: low-beta-hybrid  
**策略全称**: 日内低贝塔抗跌 + 相对强度混合策略  
**创建日期**: 2026-03-26  
**状态**: ✅ 已实现

---

## 策略概述

### 核心思想

先用**日线预筛选**挑出"天生抗跌体质"的股票（低贝塔 β<0.8），再用**5分钟高频数据**确认"今天真正逆势"的个股。

> 💡 **一句话总结**: 低贝塔防御 + 相对强度 = 回撤更小、胜率更高的逆势策略

### 与原独立强度因子的区别

| 维度 | 独立强度因子 (01) | 低贝塔混合 (03) | 优势 |
|------|------------------|----------------|------|
| **预筛选** | 无 | 日线低贝塔 (β<0.8) | 过滤高β假强者 |
| **信号粒度** | 纯5分钟 | 日线预筛 + 5分钟实时 | 更稳健 |
| **参考基准** | 板块相对收益 | 板块相对收益（仅限低β池） | 抗系统性风险 |
| **回撤** | 中等 | **更小** | 低β天然防御 |
| **持有周期** | 1-3天 | **2-5天** | 更灵活 |
| **信号数量** | 较多 | **较少（更精）** | 质量更高 |

## 计算规则（两步走）

### Step 1: 日线低贝塔预筛选

```
输入: 过去20个交易日数据
↓
计算: 个股收益率 vs 中证500收益率的贝塔(β)
↓
筛选: β < 0.8 且 抗跌次数 ≥ 8次
↓
输出: 低贝塔股票池 (通常200-400只)
```

**关键指标**:
- **贝塔(β)**: 对中证500的系统性敏感度
- **抗跌次数**: 指数下跌日个股相对收益>0的次数

### Step 2: 5分钟相对强度计分

在低贝塔池内，完全复用原独立强度因子的计分规则：

| 场景 | 板块跌幅 | 个股涨幅 | 超额收益 | 得分 |
|------|---------|---------|---------|------|
| 场景2 | <-0.5% | >0% | >1% | **+2** |
| 场景3 | <-0.5% | >0% | ≤1% | **+1** |
| 场景4 | <-0.5% | ≤0% | >1% | **+1** |

**综合得分公式**:
```
hybrid_score = (1 - beta) * 50 + raw_score * 10
```

## 风险控制

| 风控措施 | 设置值 | 说明 |
|---------|--------|------|
| **单仓上限** | 5% | 分散风险 |
| **止损线** | -4% | 低贝塔已降低波动 |
| **过滤条件** | 排除ST + 流通市值<50亿 | 避免小盘风险 |
| **最大持仓** | 8-10只 | 适度分散 |

**不适用场景**:
- 单边牛市（低贝塔池信号稀少）
- 流动性危机（系统性风险）
