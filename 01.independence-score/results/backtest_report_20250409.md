## 一、策略概述

### 1.1 策略原理

独立强度因子（Independence Score）是一种基于板块相对强度的选股因子，旨在识别在板块下跌时表现抗跌的个股。

**计分规则**：
- 当板块跌幅 < -0.5% 时：
  - 个股涨幅 > 0% → +1 分
  - 相对板块超额收益 > 1% → +1 分
- 因子值 = Σ(符合计分条件的 5 分钟区间数)

### 1.2 回测逻辑

1. **信号生成**：选取独立强度因子值 ≥ 阈值的股票作为买入信号
2. **持仓周期**：持有 N 天后卖出
3. **收益计算**：(卖出价 - 买入价) / 买入价 × 100%
4. **指标统计**：胜率、平均收益、夏普比率、最大回撤等

### 1.3 回测参数

| 参数 | 数值 |
|------|------|
| 回测区间 | 2026-02-05 ~ 2026-04-08 |
| 信号阈值 | score ≥ 0.5 |
| 持仓天数 | 5 天 |
| 数据库 | ClickHouse (tdx2db_rust) |

## 六、结论与建议

### 6.1 核心结论

#### 负面发现：
1. **策略表现不佳**：回测期间平均收益为 -1.17%，整体亏损
2. **胜率偏低**：35.68% 的胜率低于随机选择（50%）
3. **夏普比率为负**：-0.25 的夏普比率表明风险调整后收益为负
4. **多数交易亏损**：64.32% 的交易以亏损告终

#### 正面发现：
1. **部分板块表现优异**：特定板块（如 sh881006、环境治理）胜率 100%
2. **4月初反弹**：2026-04-03 信号胜率 72.1%，平均收益 1.17%
3. **存在结构性机会**：最大单笔收益 17.12%，说明策略能捕捉到强势个股

### 6.2 策略改进建议

#### 1. 参数优化（优先级：高）
- **提高阈值至 1.0**：回测显示阈值 1.0 时夏普比率转正（0.17），明显优于阈值 0.5（-0.25）
- **缩短持仓**：测试 1-3 天持仓，减少市场波动影响
- **设置止盈止损**：建议 -5% 止损、+10% 止盈

**阈值对比详细数据**：
| 指标 | 阈值 0.5 | 阈值 1.0 | 改善 |
|------|----------|----------|------|
| 信号数 | 1,435 | 122 | -91.5% |
| 平均最大回撤 | -8.17% | -6.88% | +15.8% |
| 夏普比率 | -0.25 | **0.17** | +168% |

#### 2. 增加过滤条件
```python
# 建议增加以下过滤条件：
- 流通市值 > 50 亿（排除小盘股流动性风险）
- 成交量放大（确认资金关注）
- 避免 earnings season（规避财报风险）
- 结合融资余额变化（融资加权版已支持）
```

#### 3. 组合优化
- **多因子组合**：结合动量因子、价值因子构建复合策略
- **行业轮动**：只在强势板块中选股
- **时间过滤**：避开市场大跌日（如 2026-03-12 ~ 2026-03-18）

#### 4. 仓位管理
- **凯利公式**：根据历史胜率调整仓位
- **分批建仓**：分 3 批建仓，降低择时风险
- **动态调仓**：根据市场波动率调整持仓比例

### 6.3 后续研究方向

1. **更长回测周期**：扩展至 2025 年数据，验证策略稳定性
2. **不同市场环境**：区分牛市、熊市、震荡市的表现
3. **参数敏感性分析**：测试不同阈值、持仓周期的组合
4. **与其他因子结合**：测试与低波动、高股息因子的协同效应
