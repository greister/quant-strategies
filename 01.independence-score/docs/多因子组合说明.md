# 多因子组合策略说明

## 什么是多因子组合？

### 简单比喻

想象你要选一只股票，只看一个指标就像"盲人摸象"：

| 只看一个因子 | 问题 |
|-------------|------|
| 只看**独立强度** | 可能选到弱势股里的"矮子里拔将军" |
| 只看**动量** | 可能选到已经涨完、即将回调的股票 |
| 只看**估值** | 可能选到便宜但永远不涨的股票 |

**多因子组合** = 同时看多个指标，综合打分选股

---

## 类比：公司招聘

假设你要招聘一名员工：

```
单因子选股（只看一个条件）：
- 只看学历 → 可能招到高分低能
- 只看经验 → 可能招到固步自封的老员工
- 只看面试表现 → 可能招到"面霸"

多因子选股（综合评估）：
- 学历 (20%) + 经验 (30%) + 能力测试 (30%) + 价值观 (20%)
- 这样选出来的人更靠谱
```

股票也是一样的道理！

## 本项目中的因子

### 1. 独立强度因子（已有）

**作用**: 找出**逆势抗跌**的股票

```
场景: 板块大跌时，个股不跌甚至上涨
特点: 强势股特征，市场回暖时可能领涨
问题: 可能选到弱势板块里的"相对强"
```

### 2. 动量因子（已有）

**作用**: 找出**趋势向上**的股票

```
计算: Momentum = 0.5×20日收益 + 0.3×10日收益 + 0.2×5日收益
场景: 过去涨得好的股票，未来可能继续涨
特点: 追涨策略，顺势而为
问题: 可能追到顶部
```

### 3. 估值因子（可添加）

**作用**: 找出**价格合理**的股票

```
指标: PE(市盈率)、PB(市净率)
场景: 避免买得太贵
特点: 价值投资思路
问题: 便宜的可能永远便宜
```

---

## 如何组合？

### 方法一：串联筛选（漏斗法）

```
第一步: 用独立强度因子，选出分数>0.05的股票（比如500只）
    ↓
第二步: 用动量因子，选出动量排名前100的股票
    ↓
第三步: 用估值因子，选出PE<30的股票（比如30只）
    ↓
最终买入这30只
```

**优点**: 每一步都缩小范围，精准度高
**缺点**: 可能漏掉好股票

---

### 方法二：加权打分（综合法）

```
给每只股票计算综合得分:

综合得分 = 独立强度×40% + 动量×40% + 估值×20%

例子:
- 股票A: 独立强度0.1(满分0.2)、动量15%(满分20%)、PE=25(满分50)
  → 得分 = (0.1/0.2)×40 + (15/20)×40 + (25/50)×20 = 20+30+10 = 60分

- 股票B: 独立强度0.05、动量18%、PE=15
  → 得分 = 10+36+30 = 76分 ← 选这个！
```

**优点**: 综合考量，不会漏掉某方面优秀的股票
**缺点**: 需要调权重参数

## 本项目推荐组合

### 组合1: 逆势+趋势（双因子）

```sql
-- 选取同时满足以下条件的股票:
-- 1. 独立强度 >= 0.05 (逆势抗跌)
-- 2. 动量排名在前20% (趋势向上)

SELECT 
    i.symbol,
    i.name,
    i.score as independence_score,
    m.momentum_score,
    (i.score * 0.5 + m.momentum_score * 0.5) as combined_score
FROM independence_score_daily i
INNER JOIN momentum_factor_daily m 
    ON i.symbol = m.symbol AND i.date = m.date
WHERE i.date = '2026-03-26'
  AND i.score >= 0.05
  AND m.rank_pct <= 0.2  -- 动量前20%
ORDER BY combined_score DESC
LIMIT 20;
```

**逻辑**: 
- 独立强度保证股票质地好（抗跌）
- 动量保证趋势向上（追涨）
- 两者结合 = 选到**强势市场中的强势股**

---

### 组合2: 逆势+趋势+估值（三因子）

```sql
-- 加入估值筛选
SELECT 
    i.symbol,
    i.name,
    i.score as independence_score,
    m.momentum_score,
    v.pe_ratio,
    (i.score * 0.4 + m.momentum_score * 0.4 + (50-v.pe_ratio)/50 * 0.2) as combined_score
FROM independence_score_daily i
INNER JOIN momentum_factor_daily m 
    ON i.symbol = m.symbol AND i.date = m.date
INNER JOIN valuation_daily v
    ON i.symbol = v.symbol AND i.date = v.date
WHERE i.date = '2026-03-26'
  AND i.score >= 0.05
  AND m.rank_pct <= 0.3
  AND v.pe_ratio < 40  -- 估值不太贵
ORDER BY combined_score DESC
LIMIT 20;
```

## 实际案例

假设某天有以下股票:

| 股票 | 独立强度 | 动量排名 | 估值PE | 单买独立强度? | 单买动量? | 双因子组合 |
|-----|---------|---------|--------|--------------|----------|-----------|
| A | 0.15(很强) | 后50%(弱) | 60(贵) | ✅ | ❌ | ❌ |
| B | 0.02(弱) | 前5%(很强) | 80(很贵) | ❌ | ✅ | ❌ |
| C | 0.08(强) | 前10%(强) | 25(合理) | ✅ | ✅ | ✅ |
| D | 0.06(较强) | 前15%(较强) | 30(合理) | ✅ | ✅ | ✅ |

**结果**:
- 单买独立强度 → 可能买到A（趋势弱，可能继续跌）
- 单买动量 → 可能买到B（独立强度弱，可能只是短期炒作）
- 双因子组合 → 买到C和D（质地好+趋势强）

## 为什么30天回测表现不好？

### 原因分析

```
测试期市场特征:
- 30天里28天上涨（93%上涨日）
- 几乎是大牛市

独立强度因子的问题:
- 专门找"逆势抗跌"的股票
- 但在牛市中，大家都在涨
- 抗跌的股票反而可能是弱势股（涨不动）

动量因子在这种市场的表现:
- 追涨策略在牛市中表现很好
- 强者恒强
```

### 多因子组合的作用

```
如果当时用了【独立强度+动量】双因子:

独立强度筛选:
- 去掉那些真的弱的股票
- 保留质地好的

动量筛选:
- 在质地好的里面
- 选趋势最强的

结果:
- 避免选到"弱的里面相对强的"
- 选到"强的里面最强的"
```

## 总结

| 概念 | 一句话解释 |
|-----|-----------|
| 单因子 | 只看一个指标选股 |
| 多因子 | 综合多个指标选股 |
| 串联筛选 | 一步接一步缩小范围 |
| 加权打分 | 每个因子给分，加权求和 |
| 为什么要组合 | 避免单一因子的盲点，选到更全面的好股票 |

**简单说**: 多因子就像招聘时既看学历又看能力，比只看一个维度更靠谱！
