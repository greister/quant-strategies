## 📖 什么是多因子组合？

### 一句话解释

**多因子组合 = 同时看多个指标，综合打分选股**

### 生活类比：招聘员工

| 只看一个条件 | 问题 |
|-------------|------|
| 只看学历 | 可能招到高分低能 |
| 只看经验 | 可能招到固步自封的老员工 |
| 只看面试 | 可能招到"面霸" |

**综合评估**：学历(20%) + 经验(30%) + 能力测试(30%) + 价值观(20%) = 更靠谱的人选

股票也是一样的道理！

## 🎯 本次演示的两个因子

### 1. 独立强度因子

**作用**: 找出**逆势抗跌**的股票

```
场景: 板块大跌时，个股不跌甚至上涨
特点: 强势股特征，市场回暖时可能领涨
```

### 2. 动量因子

**作用**: 找出**趋势向上**的股票

```
计算: Momentum = 0.5×20日收益 + 0.3×10日收益 + 0.2×5日收益
场景: 过去涨得好的股票，未来可能继续涨
特点: 追涨策略，顺势而为
```

## 💡 多因子组合的优势

### 组合前后对比

| 对比维度 | 单因子（独立强度） | 双因子组合 |
|---------|------------------|-----------|
| 选股范围 | 防御板块为主 | 多板块分散 |
| 上涨动能 | 弱（防御股） | 强（强势股） |
| 抗跌能力 | 强 | 较强 |
| 综合质量 | 一般 | 更好 |

### 关键发现

```
单因子问题:
- 只看独立强度 → 选到弱势股里的"相对强"（电力、燃气等防御股）
- 只看动量 → 可能追到顶部（缺乏质地验证）

双因子优势:
- 独立强度 ≥ 0.05 保证股票质地（抗跌能力）
- 动量排名前30% 保证趋势方向（上涨动能）
- 综合得分 = 独立强度×50% + 动量×50%
- 结果 = 强势市场中的强势股
```

## 🔧 如何实现多因子组合

### SQL 示例

```sql
-- 双因子组合选股
SELECT 
    i.symbol,
    i.sector,
    i.score as independence_score,
    m.momentum_score,
    m.return_20d,
    (i.score * 0.5 + m.momentum_score * 0.5) as combined_score
FROM independence_score_daily i
INNER JOIN momentum_factor_daily m 
    ON i.symbol = m.symbol AND i.date = m.date
WHERE i.date = '2026-03-26'
  AND i.score >= 0.05              -- 独立强度筛选
  AND m.rank_pct <= 0.3            -- 动量前30%
ORDER BY combined_score DESC
LIMIT 20;
```

### Python 脚本

```python
# 独立强度权重 vs 动量权重
weights = {
    'bull_market':    {'independence': 0.3, 'momentum': 0.7},  # 牛市重动量
    'neutral_market': {'independence': 0.5, 'momentum': 0.5},  # 震荡平衡
    'bear_market':    {'independence': 0.7, 'momentum': 0.3},  # 熊市重防御
}

# 计算综合得分
combined_score = (
    independence_score * weights['market_type']['independence'] +
    momentum_score * weights['market_type']['momentum']
)
```

## 📈 30天回测的启示

### 为什么30天回测表现不好？

```
市场环境:
- 30天里28天上涨（93%上涨日）
- 几乎是大牛市

独立强度因子的问题:
- 专门找"逆势抗跌"的股票
- 但在牛市中，大家都在涨
- 抗跌的股票反而可能是弱势股（涨不动）

如果当时用了【独立强度+动量】双因子:
- 独立强度筛选去掉那些真的弱的股票
- 动量在质地好的里面选趋势最强的
- 避免选到"弱的里面相对强的"
- 选到"强的里面最强的"
```
