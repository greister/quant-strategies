## 五、改进建议

### 5.1 短期优化

1. **推荐默认使用"行业独立强度(严)"**
   - 平均收益最高 (+0.59%)
   - 胜率最高 (52.9%)
   - 夏普比率最佳 (0.219)

2. **动态阈值调整**
   ```
   高波动日(>3%) → 使用严格阈值(-0.5%)
   低波动日(<1%) → 使用宽松阈值(-0.2%)
   ```

3. **组合策略**
   ```
   行业独立强度(严) + 小时VWAP强势
   双重过滤，提高胜率
   ```

### 5.2 中期优化

1. **多基准扩展**
   - 增加中证500、中证1000基准
   - 小盘/大盘差异化选股

2. **时间加权**
   - 收盘前1小时权重增加
   - 早盘和午盘差异化处理

3. **机器学习筛选**
   - 用历史数据训练有效特征
   - 预测次日涨跌概率

### 5.3 风险提示

1. **样本局限**
   - 回测仅11个交易日
   - 缺乏完整牛熊周期验证

2. **市场环境**
   - 4月初整体下跌，因子普遍亏损
   - 需增加上涨市/震荡市回测

3. **过拟合风险**
   - 严格阈值可能在其他时期失效
   - 建议分阶段验证
